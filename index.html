Who are we?

Rivi is a newly formed organization on software and technology. This is a Sri Lankan organization that aims to develop our own software in Sri Lanka. When foreign softwares are produced globally we have seen many instances where it is incompatible for the use in Sri Lanka. To overcome those defects, we need to develop software that is suitable, appropiate and more convenient for the users of Sinhala as well as english language.
